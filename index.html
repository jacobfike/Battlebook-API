Battlebook-api

A wrapper for Blizzard's Battle.net API used by Battlebook.me

View project onGitHub

Installation

$ git clone git@github.com:jacobfike/Battlebook-API

Copy API.php into your project's path.

Usage

<?php

require_once 'API.php';

$api = new Battlebook_Service_Bnet_Api(array(
    'region' => Battlebook_Service_Bnet_Api::REGION_US
));
$response = $api->getRealmStatus('Thorium Brotherhood');

?>
